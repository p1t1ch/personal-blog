Сетка из карточек превью для постов из блога. Состоит из вариации `BlogPostHead` и блока описания. Карточка полностью обёрнута в ссылку. Первая карточка имеет увеличенные размеры, т.к. является последней по времени опубликованной статьёй. Также для неё отключена transform анимация, кажущаяся там неуместной. В приложении эта сетка обёрнута в контейнер, не позволяющий более 3 карточек в ряд.
