Подзаголовок на главной странице. Заполняется 3 случайными тегами из списка. Теги выбираются при загрузке страницы, так что нет возможности добавить эту информацию в html на этапе сборки - по этой причине подзаголовок скрывается на серверном рендере.

По правилам русского языка вместо предлого "О" пишется предлог "Об", если следующее за ним слово начинается с букв "АИУОЭ". Компонент учитывает этот момент, проверяя первую букву первого тега.
