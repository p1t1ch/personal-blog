Контейнер основной части страницы: задаёт ограничения по ширине и отступы. Вариация для страницы поста из блога имеет уменьшенные размеры для удобства восприятия текстового контента.

Компонент не включён в `Layout`, т.к. его вставка для страниц блога идёт не на верхнем уровне, а уже внутри **article**, чтобы позволить заголовочной части занимать полную ширину.
